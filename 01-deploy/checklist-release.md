# Checklist de Release

## Antes do Deploy
- [ ] PR aprovado
- [ ] QA validado
- [ ] Migration revisada
- [ ] Rollback definido
- [ ] Feature flag configurada

## Durante
- [ ] Pipeline verde
- [ ] Sem erro de migration
- [ ] Serviço online

## Após
- [ ] Smoke test executado
- [ ] Monitoramento ativo
- [ ] Comunicação enviada
